# START HERE — Подготовка к воркшопу

> Время: ~30 минут. Сделайте за 2-3 дня до воркшопа.
> На самом воркшопе мы не будем ждать установку.

---

## Что понадобится

- Ноутбук (macOS, Windows или Linux)
- 8GB+ RAM, 10GB свободного места
- Стабильный интернет
- Зарядка (AI coding быстро сажает батарею)

---

## Шаг 1. Создайте аккаунты

Все бесплатные, кроме Claude.

| # | Сервис | Зачем | Ссылка |
|---|--------|-------|--------|
| 1 | **GitHub** | Хранение кода | [github.com/signup](https://github.com/signup) |
| 2 | **Claude Pro или Max** | Подписка для AI-агента | [claude.ai](https://claude.ai) |

> Vercel и Supabase входите **через GitHub** — так проще и не нужно запоминать отдельный пароль.

---

## Шаг 2. Установите Conductor

Conductor — это приложение для работы с AI-агентом. Думайте о нём как о "рабочем столе разработчика".

1. Скачайте Conductor: [conductor.build](https://conductor.build)
2. Установите (macOS: перетащите в Applications)
3. Откройте Conductor
4. Нажмите **Sign In** — войдите через GitHub

---

## Шаг 3. Откройте этот репозиторий в Conductor

1. В Conductor нажмите **Open Project** (или Quick Start)
2. Выберите папку, куда вы скачали этот репозиторий
3. Дождитесь пока Conductor откроет проект

> Если вы ещё не скачали репозиторий — Conductor может сам клонировать его с GitHub.

---

## Шаг 4. Запустите автоматическую настройку

В чате с Claude (внутри Conductor) напишите:

```
Прочитай файл workshop/prework/PREWORK_INSTRUCTIONS.md и выполни все шаги по порядку.
```

**Что произойдёт:**
- Claude прочитает инструкции и начнёт устанавливать всё необходимое
- Иногда он будет просить вас что-то подтвердить (например, авторизовать GitHub в браузере) — просто следуйте его указаниям
- В конце он спросит, какие типы проектов вы планируете делать, и доустановит нужное
- По завершении он создаст файл `setup-report.md` с отчётом

Весь процесс занимает ~15-20 минут.

---

## Шаг 5. Отправьте отчёт

Когда Claude закончит, в корне проекта появится файл **`setup-report.md`**.

Отправьте его ведущему воркшопа — так мы убедимся что у вас всё готово.

---

## Что-то пошло не так?

Если Claude не смог что-то установить — откройте файл `workshop/fallback/manual-checklist.md`. Там пошаговая инструкция для ручной установки.
