# START HERE — Подготовка к воркшопу

> Время: ~30 минут. Сделайте за 2-3 дня до воркшопа.
> На самом воркшопе мы не будем ждать установку.

---

## Шаг 1. Создайте аккаунты (5 мин)

Все бесплатные, кроме Claude.

| # | Сервис | Зачем | Ссылка |
|---|--------|-------|--------|
| 1 | **GitHub** | Хранение кода | [github.com/signup](https://github.com/signup) |
| 2 | **Claude Pro или Max** | Подписка для AI-агента | [claude.ai](https://claude.ai) |
| 3 | **Vercel** | Хостинг сайта (бесплатно) | [vercel.com](https://vercel.com) — войти через GitHub |
| 4 | **Supabase** | База данных (бесплатно) | [supabase.com](https://supabase.com) — войти через GitHub |

> Vercel и Supabase входите **через GitHub** — так проще и не нужно запоминать отдельный пароль.

---

## Шаг 2. Откройте терминал

- **macOS:** нажмите `Cmd + Пробел`, наберите "Terminal", нажмите Enter
- **Windows:** установите WSL2 (`wsl --install` в PowerShell от администратора), затем откройте Ubuntu из меню Пуск
- **Linux:** `Ctrl + Alt + T`

---

## Шаг 3. Установите Claude Code и запустите настройку (15-20 мин)

Скопируйте и вставьте эту команду в терминал:

**macOS / Linux / WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows (PowerShell):**
```powershell
irm https://claude.ai/install.ps1 | iex
```

Проверьте что установилось:
```bash
claude --version
```

Теперь запустите Claude Code:
```bash
claude
```

При первом запуске он попросит войти в аккаунт Claude — следуйте инструкциям на экране. Когда откроется чат, вставьте этот текст:

```
Склонируй репозиторий https://github.com/rrrtem/ai-coding и выполни все шаги из файла workshop/prework/PREWORK_INSTRUCTIONS.md по порядку.
```

**Что произойдёт:**
- Claude скачает репозиторий с инструкциями
- Начнёт устанавливать базовые инструменты (Node.js, Git, GitHub CLI, Vercel, Supabase и др.)
- Будет объяснять каждый шаг: что ставится, зачем, как связано с остальным
- Иногда попросит вас что-то подтвердить (например, авторизовать GitHub в браузере) — просто следуйте его указаниям
- Спросит, какие проекты вам интересны (сайты, боты, мобильные приложения) — и доустановит нужные инструменты
- По завершении создаст файл `setup-report.md` с отчётом

---

## Шаг 4. Отправьте отчёт

Когда Claude закончит, в корне проекта появится файл **`setup-report.md`**.

Отправьте его ведущему воркшопа — так мы убедимся что у вас всё готово.

---

## Что-то пошло не так?

Если Claude Code не удаётся установить или он не справился с настройкой — откройте `workshop/fallback/manual-checklist.md`. Там пошаговая инструкция для ручной установки.
