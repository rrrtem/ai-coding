## 2.3 Telegram-бот

Нужен Python.

Проверь: `python3 --version`

Если не установлен:

**macOS:**
```bash
brew install python
```

**Linux:**
```bash
sudo apt install -y python3 python3-pip python3-venv
```

Проверь: `python3 --version` (3.10+) и `pip3 --version`

Сообщи:
> Python установлен. Библиотеки для бота (python-telegram-bot, supabase) будут установлены уже в конкретном проекте, не глобально — так правильнее.

## 2.4 Мобильное приложение (iOS / Android)

Нужны Expo CLI и EAS CLI.

```bash
npm install -g expo-cli eas-cli
```

Проверь:
```bash
expo --version
eas --version
```

**Для iOS-разработки на macOS** — проверь что Xcode установлен:
```bash
xcode-select -p
```

Если нет — скажи:
> Для тестирования iOS-приложений на симуляторе нужен Xcode. Установи его из App Store (это бесплатно, но весит ~12GB). Можно сделать это позже. Для Android — можно тестировать через Expo Go на телефоне без дополнительных установок.

Сообщи:
> Expo CLI и EAS CLI установлены. Для тестирования можно использовать приложение Expo Go на телефоне — скачай его из App Store / Google Play.
